##### python3 -m pip install ansible boto3 botocore --user #####
"--user" installs packages only for your user account
Packages go into something like:
~/.local/lib/python3.x/site-packages
It does not touch system Python or distro-managed packages (apt, dnf, etc.)
👉 So you avoid the classic “pip broke my OS Python” problem. Good practice.



One smart move before going further (strongly recommended)

Freeze this working setup so you can always come back to it:

##### pip freeze > requirements.txt #####

If anything explodes later, you’ll be able to recover in 30 seconds.
